§03

lore // giro_inesperado

narrativas estocásticas sobre la fricción de crear, codificar y perderse entre madrid y silicon valley.

capítulo 01 // el callejón de madrid

el aire de madrid a las cuatro de la mañana huele a ozono y asfalto frío. las luces de los sintetizadores iluminaban mi habitación con una luz azul hibonita muy densa. el ordenador escupía miles de líneas de logs redundantes mientras cargaba el preloader del sistema.

"la entropía acumulada no se destruye, solo se transforma en ruido sónico."

capítulo 02 // el puente transatlántico

un correo de silicon valley rompió el bucle sónico. de repente, los cables de cobre del sintetizador modular se cruzaron con las peticiones de red asíncronas de un servidor en la nube. el portfolio dejó de ser una galería muerta para convertirse en un organismo vivo.

capítulo 03 // la mesa de control

allí estaba yo, ante una pantalla que renderizaba buffers de audio mientras el silicon oracle firmaba la validez de cada muestra. el ruido analógico se convirtió en código determinista de exergía pura.
